# Coinsight

## 프로젝트 개요
뉴스와 이슈에 민감한 암호화폐 투자를 돕기 위해, AI를 활용하여 암호화폐 관련 뉴스를 자동으로 수집·분석하고, 각 코인에 미치는 영향(호재/악재)을 예측하여 투자자들의 의사결정을 지원하는 플랫폼

## 프로젝트 일정
프로젝트 기간 2025/12/29 ~ 2026/03/29 (13주)

- 기획/배포 3주 (2025/12/29 ~ 2026/01/18)
- 개발 8주 (2026/01/19 ~ 2026/03/15)
- 프로젝트 정리 2주 - 테스트/산출물 정리 (2026/03/16 ~ 2026/03/29)

## 요구사항 정의
- 메인 페이지
    - 코인 시세 조회 기능
        - 거래대금이 높은 순으로 코인을 조회할 수 있다.
        - 국내시세/해외시세로 구분해서 코인을 조회할 수 있다.
        - 코인명, 가격, 등락폭, 거래대금을 코인을 조회할 수 있다.
        - 준 실시간(1초 이상 1분 이하)간격으로 가격이 화면에서 업데이트 된다.
        - 검색된 코인을 클릭하여 “코인별 상세 페이지”로 이동할 수 있다.
    - 코인 검색 기능
        - 조회 결과로 코인의 심볼, 코인의 이름, 코인의 약어, 코인의 통화류를 볼 수 있다.
        - 코인 명으로 검색할 수 있다.
        - 코인 약어로 검색할 수 있다. ex) ETH
        - 검색된 코인을 클릭하여 “코인별 상세 페이지”로 이동할 수 있다.
        - 거래대금 내림차순으로 20개의 코인이 노출된다.
    - AI 뉴스 분석 결과 조회 기능
        - 뉴스의 분석결과 (호재/악재)뱃지, 뉴스의 제목, 뉴스 출처, 뉴스 발행 시점, 관련 코인 뱃지, 추측 신뢰도를 볼 수 있다.
        - 검색된 뉴스를 클릭하여 “뉴스별 상세 페이지”로 이동할 수 있다.
    - 뉴스 검색 기능
        - 조회 결과로 뉴스의 분석결과 (호재/악재)뱃지, 뉴스의 제목, 뉴스 출처, 뉴스 발행 시점, 관련 코인 뱃지, 추측 신뢰도를 볼 수 있다.
        - 뉴스의 제목으로 검색할 수 있다.
        - 뉴스의 내용으로 검색할 수 있다.
        - 검색된 뉴스를 클릭하여 “뉴스별 상세 페이지”로 이동할 수 있다.
        - 최신순으로 20개의 기사가 노출된다.
- 코인별 상세 페이지
    - 상단
        - 코인의 이름과, 통화류, 약어, 최종 업데이트 일시, 현재가, 1시간 등락 폭, 24시간 거래대금을 조회할 수 있다.
    - AI 예측 히스토리
        - 1시간, 3시간, 6시간, 12시간, 24시간 간격을 선택하여, 시간별 예측의 실제 결과를 볼 수 있다. (예측~간격~결과)
        - 예측 시점, AI 예측(호재/악재), 실제 결과(등락폭), 예측 결과(예측 성공/예측 실패), 사용된 기사를 볼 수 있다.
        - 예측에 사용된 기사를 조회할 수 있다.
            - 기사의 호재/악재 예측 뱃지, 기사 제목, 기사 출처, 기사 시간을 조회할 수 있다.
            - 예측이 지난 후, 기사의 실제 예측 결과가 문장으로 표현되어 볼 수 있다.
            - 기사를 눌러서 기사 상세 페이지로 이동할 수 있다.
    - AI 예측 누적 통계
        - 총 예측 횟수, 예측 성공 횟수, 총 성공률, 예측 간격별 예측 성공률을 조회할 수 있다.
        - 예측 간격별 예측 성공률을 조회할 수 있다.
            - 1시간, 3시간, 6시간, 12시간, 24시간 간격에 대한 예측 성공률을 볼 수 있다.
    - 코인 차트
        - 추후 정의
- 기사별 상세 페이지
    - 기사 메인
        - 기사의 예측 결과(호재/악재)뱃지를 볼 수 있다.
        - 기사 제목을 볼 수 있다.
        - 발행 시간(2시간 전 등), 출처, 예측 신뢰도를 볼 수 있다.
        - 기사 본문을 볼 수 있다.
        - 기사에 사용된 긍정요소 해쉬태그, 부정요소 해쉬태그 뱃지를 볼 수 있다.
    - 실제 결과 검증 (과거 기사)
        - 기사 보도 후, 실제로 연관된 코인이 오르내린 경우, 해당 코인이 x시간후 실제로 오른내린 것을 띄워준다.
    - 영향 받는 코인
        - 코인명, 예상 영향(호재/악재), 영향 정도를 볼 수 있다.
    - AI 종합 분석
        - 추후 정의
- 호재/악재 맞추기 게임 진행 페이지
    - 게임 페이지에 진입하면 초기화되며 바로 게임이 시작된다.
    - 상단
        - 게임 제목, 간략한 게임 설명, 현재 점수를 볼 수 있다.
    - 문제
        - 문제 기사의 AI 예측 시점의 차트를 볼 수 있다.
        - 문제 기사의 발행 일자, 관련 코인목록, 제목, 내용, 문제 글(이 뉴스는 X코인에 호재일까요, 악재일까요?)를 볼 수 있다.
        - 호재, 악재 버튼이 존재하며 하나를 클릭하면 선택이 제출된다.
            - 정답인 경우, 정답이 표기되고 현재 점수가 오르며 다음 문제로 넘어간다.
            - 오답인 경우, 게임이 종료되고 현재 점수가 표시되며, 닉네임을 입력할 수 있다.
                - 닉네임을 입력하면 리더보드에 등록되고, 리더보드 화면으로 이동된다.
- 호재/악재 맞추기 게임 리더보드
    - 상단
        - 리더보드 제목과 간략한 설명을 조회할 수 있다.
    - 본문
        - 순위, 닉네임, 점수, 달성 날짜를 페이징 조회할 수 있다.
